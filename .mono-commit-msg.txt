Guided Assign-learners wizard (org -> class -> courses -> enrol)

New AssignWizard.tsx: one pop-up that walks the whole allocation pipeline in order - pick org learners (or add a new learner to the org inline), choose or create a class, assign courses, then review and enrol. Every write MERGES into the class's existing roster/courses (never replaces) and materialises real enrolments via the existing endpoints. Launchable org-wide from the org PageHeader (Assign learners button on every org sub-page) and from a class via a preselected wizard on PartnerClassDetail. Removed the now-duplicate People-tab button. Frontend-only; no backend change.
