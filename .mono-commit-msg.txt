Full account Manage drawer: reset, temp password, suspend, archive, delete, reassign org

The Manage drawer was near-empty for partner admins. Rebuilt it with a full, role-safe action set usable by BOTH partner admins and super admins: email a reset / resend-invite link, set a temporary password, reassign the account's organisation, suspend/reactivate, archive/restore, and a soft (recoverable) delete behind a two-step confirm. Soft lifecycle uses two new nullable columns archived_at/deleted_at (added to schema + healed idempotently at boot in dbHardening, so no enum migration and every full-row select stays valid). New endpoints: POST /partners/:pid/members/:userId/lifecycle (suspend|reactivate|archive|restore|delete) and .../organisation (reassign) - both gated to super OR the partner's own admin, never self, and a partner admin can't touch admin tiers. The Accounts roster now hides archived/removed rows behind a 'Show archived & removed (n)' toggle with Archived/Removed badges. A partner admin still can't manage another admin or their own row. Files: schema/users.ts, dbHardening.ts, routes/platform.ts, routes/partners.ts, pages/partner/PartnerAccounts.tsx.
