Editable module Video section: add/upload a video to a module (ModuleViewer) (ASCII-only)

The module Video section is now editable for instructors: a ModuleVideoAdmin panel lets you set a title and attach a video by URL, or Upload file (base64 -> /learning/content/upload -> Supabase Storage, returns a URL). Saving PATCHes the video beat's videoUrl, creating a video beat (POST /modules/:id/beats type=video) if the module has none. Learners then see the video in this section. Until Supabase is configured, pasting a hosted URL works; file upload shows a clear message. Panel shows in both the empty and populated states. Icons verified imported.
