In-course Case studies tab (list/attach/detach via gradebook) (ASCII-only)

- Backend: new GET /courses/:courseId/cases (cases.ts) returns the case studies attached to a course by resolving its gradebook case columns to case titles/status + the gradebook item id (for detach).
- Frontend: CourseDetail gains a Case studies tab (CourseCasesTab) mirroring the Activities tab - lists attached cases, Attach existing (POST gradebook-items sourceType=case), New case study (to the Cases author), Open (case builder), and Detach (DELETE /gradebook-items/:id; the case itself is not deleted). Uses existing endpoints.

VALIDATION: esbuild clean on CourseDetail.tsx and cases.ts.
